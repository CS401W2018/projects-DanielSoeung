CSS

1. Explain the difference between inline, embedded, and external CSS.

a. Inline is the 2nd priority and has a style attribute, but it is often discouraged to use.

b. Embedded is the 3rd priority and has a style tag. Embedded also has a selector in what we want to modify.

c. External CSS is making the style on a seperate file and then applying it your webpage

2. Explain why you should only use external CSS.

a. Seperation of Concern Helps with SEO, helps browers display our content conrrectly

b. A stylesheet can be used for multiple websites you make

3. What is a selector used for?

a. A selector is used to modify an element on your webpage. You can change the color of text or the page itself
