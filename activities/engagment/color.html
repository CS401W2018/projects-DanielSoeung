What is the difference between an ID and a Class?

Instead of using a "#" in CSS for ID the Class uses a "." to call the method/object. ID you can only be used once. A Class can name more than one tag and you can also put multiple classes in a tag

What is the difference between a SPAN tag and a DIV tag?

The main difference between the two is that a DIV tag will force line breaks. While a SPAN does not.

What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

There are no difference between those tags. But google will interpret your webpage by what you put in your DIV, HEADER, MAIN and FOOTER.
