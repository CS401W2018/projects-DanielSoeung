Why is the viewport meta tag important?

Makes sure that you page is displayed and structured properly.

How can you adjust the CSS for printing?

Link it to a seperate CSS file and make sure to include the media tag for each. One for the screen and the other for print.

When would you want to adjust the CSS for printing?

When you intend for the page to be printed.
