How can you adjust the CSS for printing?

Link it to a seperate CSS file and make sure to include the media tag for each. One for the screen and the other for print.

When would you want to adjust the CSS for printing?

When you intend for the page to be printed.

Why do we need a viewport meta tag?

Makes sure that you page is displayed and structured properly.

When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

It will take the most recent one which would be the second one in this case.

How do you change the CSS for a web page displayed on a phone?

You can use the media in CSS and limit the max width for the phone page to take affect.
