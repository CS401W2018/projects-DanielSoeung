Questions

Explain the difference between an absolute link and a relative link. Why would you use relative links?

An absolute link is a link that contains all the information required to access the webpage. Relative links are links within your own domain. Which means that the link has to have the same domain as the page you are linking it from. Instead of usisng everything needed as in absolute you can just use the file name.

List the 3 rules with relative links

  1. Use the file name
  2. To up to a directory use ../file name
  3. Start at the root "/"

Do links always need to be inside a NAV tag?

No links do not always have to be inside the NAV tag. They can be outside the nav as well.
