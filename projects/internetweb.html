Without this complex and amazing system computers will not be able to communicate effectively.

How are the Internet and Web different?

Definition: The internet is how the computers are able to communicate to each other. In contrast the web is allows the content to be accessible to you as a user looking to access the information from the internet system.

ASCII and Unicode

Definition: ASCII is an encoding format where texts or letters are encoded into numeric values or binary so that the computer can understand it. The Unicode is an extension of ASCII it includes more special characters or letters that would not be available in ASCII.

What is UTF-8

Definition: UTF-8 is a character encoding format where it encodes characters in 8 bit bytes

URL

Definition: The URL is the address or the directory for the website. When you enter a link for a certain site the URL acts as the address for that site so you can access it.

Domain Name

Definition: The Domain Name is the name of who owns the name of the website. If you type in "www.apple.com", apple is the owner of the domain name.

HTTP

Definition: HTTP allows data to be transmitted between the web brower and the web. Once it communicates between each other the web is then visble for the web broswer.

HTTPS

Definition: HTTPS works the same as HTTP by allowing data transmission. Except this protocol is a much more secuere method.

IP Address

Definition: An IP Address is a unique set of number assigned to a device. Each device has its own unique to it only and that is how the device is identified.

DNS

Definition: A DNS is essentially a system that converts domain names into IP addresses so you can load the website on the web.

W3C Recommendations

Definition: W3C Recommendations is a set of guidline to ensure proper web development standards. It aims to promote good and consistent practices among developers.
